## 功能需求
### 界面需求
#### 后台管理
- 管理主页，记录最新文章，最新留言，最近日志等
  - 最近日志记录登录IP，地址，操作等
  - 记录一天的访问量
- 发布文章
  - 使用markdown编辑器，支持插入代码，插入图片等功能
  - 能够给文章添加缩略图。
  - 可将文章存为草稿或者发布
  - 文章可选择分类和标签，自定义url
  - 文章可控制是否允许评论
